# 2798. Number of Employees Who Met the Target
- Method 1

    Give-away.

    | |   Complexity  |
    | ----------- | ----------- | 
    |  Memory     | O(1) | 
    |      Time       |  O(n) | 


    Solution:

    ``` h

    class Solution {
    public:
        int numberOfEmployeesWhoMetTarget(vector<int>& hours, int target) {
            int ans = 0;
            for(auto t : hours) ans += t >= target;
            return ans;
        }
    };

    ```
