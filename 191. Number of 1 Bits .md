#  191. Number of 1 Bits
- Method 1

    Just manipulate bits with >> and &.

    | |   Complexity  |
    | ----------- | ----------- | 
    |  Memory     | O(1) | 
    |      Time       |  O(1) | 


    Solution:

    ``` h

    class Solution {
    public:
        int hammingWeight(uint32_t n) {
            int ans = 0;
            for(int i = 0; i < 32; i++) ans += n >> i & 1;
            return ans;
        }
    };

    ```
