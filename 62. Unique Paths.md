# 62. Unique Paths
- Method 1

    Use dp. Init d[m*n] with 1.
    $$dp[i, j] = dp[i-1, j] + dp[i, j-1]$$
    To save memory, we only need d[n], and
    $$ d[n] += d[n-1] $$

    <br>

    | |   Complexity  |
    | ----------- | ----------- | 
    |  Memory     | O(min(m, n)) | 
    |      Time       |  O(m*n) | 


    Solution:

    ``` h

    class Solution {
    public:
        int uniquePaths(int m, int n) {
            if(m < n) return uniquePaths(n, m);
            vector<int> cur(n, 1);
            for(int i = 1 ; i < m; i++) {
                for(int j = 1; j < n; j++) {
                    cur[j] += cur[j-1];
                }
            }
            return cur[n-1];
        }
    };

    ```

- Method 2

    This is math solution. We can prove that the number of f(m, n) = $C^{m-1}_{m+n-2}$, and there is also a [detailed explanation](https://leetcode.com/problems/unique-paths/solutions/2948973/clean-c-maths-solution/) for it by me.

    | |   Complexity  |
    | ----------- | ----------- | 
    |  Memory     | O(1) | 
    |      Time       |  O(min(m, n)) | 


    Solution:

    ``` h

    class Solution {
    public:
        int uniquePaths(int m, int n) {
            if(m <= 1 || n <= 1) return 1;
            if(m < n) return uniquePaths(n, m);
            m--;
            double ans = 1;
            while(--n) {
                ans *= (m+n);
                ans /= n;
            }
            return round(ans);
        }
    };

    ```
