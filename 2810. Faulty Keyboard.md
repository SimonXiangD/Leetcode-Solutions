    Give-away problem.

    | |   Complexity  |
    | ----------- | ----------- | 
    |  Memory     | O(n) | 
    |      Time       |  O($n^2$) | 


    Solution:

    ``` h

    class Solution {
    public:
        string finalString(string s) {
            string ans;
            for(auto t : s) {
                if(t == 'i') reverse(ans.begin(), ans.end());
                else ans += t;
            }
            return ans;
        }
    };

    ```
