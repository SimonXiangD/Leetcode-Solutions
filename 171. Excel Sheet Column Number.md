# 171. Excel Sheet Column Number 
- Method 1

    Easy.

    | |   Complexity  |
    | ----------- | ----------- | 
    |  Memory     | O(1) | 
    |      Time       |  O(n) | 


    Solution:

    ``` h

    class Solution {
    public:
        int titleToNumber(string columnTitle) {
            int ans = 0;
            for(auto t : columnTitle) {
                ans = ans * 26 + int(t - 'A') + 1;
            }
            return ans;
        }
    };

    ```
