# 136. Single Number 
- Method 1

    Use XOR.

    | |   Complexity  |
    | ----------- | ----------- | 
    |  Memory     | O(1) | 
    |      Time       |  O(n) | 


    Solution:

    ``` h

    class Solution {
    public:
        int singleNumber(vector<int>& nums) {
            int t = 0;
            for(auto a : nums) t ^= a;
            return t;
        }
    };

    ```
