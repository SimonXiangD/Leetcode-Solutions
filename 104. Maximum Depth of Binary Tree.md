# 104. Maximum Depth of Binary Tree 
- Method 1

    Just recursion.

    | |   Complexity  |
    | ----------- | ----------- | 
    |  Memory     | O(1) | 
    |      Time       |  O(n) | 


    Solution:

    ``` h

    class Solution {
    public:
        int maxDepth(TreeNode* root) {
            if(!root) return 0;
            return 1 + max(maxDepth(root->left), maxDepth(root->right));
        }
    };

    ```
