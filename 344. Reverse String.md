# 344. Reverse String
- Method 1

    One-liner.

    | |   Complexity  |
    | ----------- | ----------- | 
    |  Memory     | O(1) | 
    |      Time       |  O(n) | 


    Solution:

    ``` h

    class Solution {
    public:
        void reverseString(vector<char>& s) {
            for(int i = 0, j = s.size()-1; i < j; i++, j--) swap(s[i], s[j]);
        }
    };

    ```
