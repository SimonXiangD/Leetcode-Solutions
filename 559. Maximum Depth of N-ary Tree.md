# 559. Maximum Depth of N-ary Tree  
- Method 1

    Use dfs.

    | |   Complexity  |
    | ----------- | ----------- | 
    |  Memory     | O(1) | 
    |      Time       |  O(n) | 


    Solution:

    ``` h

    /*
    // Definition for a Node.
    class Node {
    public:
        int val;
        vector<Node*> children;

        Node() {}

        Node(int _val) {
            val = _val;
        }

        Node(int _val, vector<Node*> _children) {
            val = _val;
            children = _children;
        }
    };
    */

    class Solution {
    public:
        int maxDepth(Node* root) {
            if(!root) return 0;
            int ans = 1;
            for(auto t : root->children) ans = max(ans, 1 + maxDepth(t));
            return ans;
        }
    };

    ```
